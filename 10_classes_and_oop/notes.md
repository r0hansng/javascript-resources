# Javascript and Classes

## OOP

- OOP is a programming paradigm that uses objects and classes to structure code
- It is a way to organize code and create reusable and modular code

## Object 
- Collection of properties and methods

## Class
- Blueprint for creating objects
- Defines the properties and methods that the objects will have
